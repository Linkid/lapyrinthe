Introduction

A propos de Lapyrinthe


Lapyrinthe est un petit jeu de labyrinthe dans lequel le joueur devra touver la sortie tout en rencontrant plus ou moins d'obstacles sur son chemin suivant le niveau choisi.
Ce jeu est développé en Python (2.4 et 2.5) avec Tkinter comme bibliothèque graphique qui est inclus avec Python. En outre, ce jeu est multiplateforme (non testé sous MacOS) ce qui permet une accessibilité meilleur..

Où l'obtenir


Vous pourrez obtenir Lapyrinthe sur Google Code.
En ce qui concerne les traductions, elles sont disponibles dans les dernières releases mais les versions les plus récentes sont présentes sur Launchpad..

A propos de ce document


Ce document est disponible dans le format HTML et en français uniquement (pour l'instant). La dernière version est toujours disponible sur Google Code.

Si vous souhaitez participer à ce projet qu'est Lapyrinthe, veuillez regarder la section Contribuer à ce projet.

Installation

Requis


Pour compiler vous-même Lapyrinthe, vous aurez besoin de Python (>= 2.4, les autres n'ont pas encore été testé), de quelques images et d'autres fichiers contenus dans l'archive téléchargée. Python est quant à lui disponible sur le site officiel Python.org.
Cependant, si vous souhaitez utiliser Python n'importe où, et en particulier sur des machines publiques, alors nous vous recommandons Portable Python qui devrait vous plaire...

A partir des sources


Lapyrinthe est un programme écrit en Python. Donc il ne se compile pas. De plus, il ne s'installe pas non plus. Alors comment s'en servir ? Tout simplement en téléhargeant les sources (voir Où l'obtenir).
Ensuite, il vous suffira d'executer le ficher inter.py. Pour cela, veuillez consulter la section Commencer.

A partir de l'executable (Windows)


Un installateur n'est pas prévu pour le programme Lapyrinthe. Il vous faudra donc télécharger la version Windows du programme puis consulter la section Commencer où l'on parle du fichier lapy.exe.

Utilisation

Commencer


Vous pouvez lancer le programme Lapyrinthe de plusieurs manières :


  • Soit à partir des sources en lançant le fichier interf.py.
  • Soit à partir de la version executable en lançant le fichier lapy.exe.

Si vous souhaitez lancer le programme à partir des sources, alors ouvrez un terminal et tapez-y ces quelques lignes :



 $ cd /chemin/vers/lapyrinthe/
 $ python interf.py


Néanmoins, si vous souhaitez changer la langue du programme (ici, en danois), alors vous remplacez la dernière ligne par celle-ci :



 $ LANG=da python interf.py


Si ouvrir un terminal ne vous convient pas, alors il vous reste la solution de l'executable Windows. Et la démarche sera d'ailleurs moindre puisque'il vous suffira de cliquer sur le fichier lapy.exe.
Cependant, la langue du programme sera la même que celle de votre système.

Général


Démarrage


Losrque vous lancerez Lapyrinthe, une fenêtre apparaîtra en plus d'un terminal. Surtout, si le terminal s'ouvre, ne le fermez pas, sinon l'application se fermera. Et si aucun terminal n'apparaît, alors ne vous inquiétez pas, ce n'est pas important. Ce terminal ne fera afficher que quelques informations comme le nombre de déplacements effectués à chaque mouvement.
La fenêtre contient différents menus détachables (Fichier, Aide), mais également un graphique blanc où sera affiché les lapyrinthes. Vous pourrez si vous le souhaitez détacher les menus pour les séparer de la fenêtre principale. Pour cela, cliquez sur les pointillés qui apparaissent au dessus du menu.
La taille de la fenêtre sera automatiquement adaptée à la taille de votre écran. Donc pas de surprise, la fenêtre ne sera pas en pleine écran dès le lancement.

Ouvrir un lapyrinthe


Après avoir lancé le programme, allez dans le menu Fichier, puis cliquez sur Ouvrir. Selectionnez ensuite le lapyrinthe que vous voulez résoudre qui devra être un fichier texte (.txt), et ouvrez-le. Il s'affichera alors dans le graphique du milieu et vour pourrez donc commencer.

Objectif


Une fois un lapyrinthe ouvert, un pion rouge (vous) apparaîtra à l'entrée du lapyrinthe sur une flèche rouge dirigée vers l'intérieur : c'est l'entrée.
Votre mission consistera alors à sortir de ce lapyrinthe en fesant le moins de déplacement possible. Cette sortie est symbolisée par une autre flèche rouge qui sera pointé vers l'extérieur.
Une fois la sortie atteinte, un message vous informera du nombre de mouvement total que vous avez fait. Et ne soyez pas surpris de ne plus pouvoir bouger votre pion après, c'est tout à fait normal.

Se déplacer


Pour se déplacer dans les lapyrinthes, vous devrez utiliser les flèches directionnelles de votre clavier. Cependant, d'autres moyens vous permettent de vous déplacer :


  • En utilisant les téléporteurs.
  • En allant sur un propulseur.

Toutefois, même si certains objets vous permettent de franchir les murs, vous ne pourrez pas traverser ces derniers à l'aide des flèches directionnelles.

Enfin, des flèches bleues seront placées sur votre chemin et vous ne pourrez allez que dans la direction qu'elles indiquent : ceux sont les sens uniques.

Lapyrinthes


Lapyrinthèque


La lapyrinthèque est le dossier où sont stockés tous les lapyrinthes.

Niveaux/Difficulté


Les lapyrinthes sont répartis par niveau de difficulté dans la lapyrinthèque. Il en existe 5 (cinq) classés par ordre croissant (du A au E):


  • Le niveau A (facile): on rentre dans le labyrinthe par la flèche qui mène à l’intérieur ; il faut en sortir par la flèche qui mène à l’extérieur. On ne peut pas se déplacer en diagonale et il est interdit de passer au travers des murs (pour ceux qui auraient envie de jouer aux fantômes).
  • Les niveaux B, C et D (moyen) : À partir du niveau B apparaissent des cases spéciales :
    • les téléporteurs,
    • les sens uniques,
    • les cases retour à la case départ.
  • Le niveau E (difficile): on trouve deux types de cases supplémentaires :
    • les propulseurs,
    • les cases stop.

Nombre de lapyrinthes disponibles


À ce jour (le 07/07/08), la lapyrinthèque compte trois lapyrinthes. Mais se nombre augmentera dans quelque temps, dès que les créateurs trouveront le temps de les numériser... De plus, le fichier cpt_lapy.txt vous indiquera le nombre de lapyrinthes disponibles dans la lapyrinthèque.

Nomenclature


Pour tous les items ayant quatre possibilités, voici l'ordre dans lequel elles sont données : Gauche Droite Haut Bas. Et l'ordre pour celles des mur est : Vertical Horizontal
Tous les caractères données sont séparés d'une virgule (,).
Il n'y a aucun ordre prédéfini en ce qui concerne les téléporteurs. Le 1 va envoyer le pion sur le 1 le plus proche. Donc si vous avez ceci : 1.1.1.1, alors il y aura deux pairs : les deux premiers 1, puis les deux suivants. On constate donc que les mêmes caractères peuvent être utilisés pour plusieurs pairs !
